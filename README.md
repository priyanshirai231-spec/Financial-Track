# FinTrack 💰

FinTrack is a full-stack expense tracking web application built using the MERN stack. It helps users manage their expenses, monitor spending, and visualize financial data through a clean and responsive dashboard.

---

## 🚀 Features

- User Authentication (Login/Register)
- Add, Edit, and Delete Expenses
- Expense Categorization
- Dashboard Analytics
- Responsive UI with Tailwind CSS
- REST API Integration
- MongoDB Database Support
- JWT Authentication
- Secure Backend Architecture

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
